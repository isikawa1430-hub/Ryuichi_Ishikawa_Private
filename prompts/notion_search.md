# Notion情報検索テンプレート

## 使い方
「Notionで〇〇を調べて」「〇〇プロジェクトのページを見せて」などと指示してください。

## エージェントの実行手順

1. `notion-search` でキーワード検索
2. 関連ページが複数ある場合はタイトル一覧を提示してユーザーに選択を促す
3. `notion-fetch` でページ内容を取得・要約
4. 更新が必要な場合はユーザーに確認してから `notion-update-page` を実行

## 検索のヒント

- プロジェクト名・案件名で検索
- 「会議メモ」「議事録」「TODO」などのキーワードで絞り込み
- 特定の日付範囲がある場合はその旨を伝える

## 出力フォーマット

```
検索結果：「[検索キーワード]」

見つかったページ：
1. [ページタイトル] — 最終更新: YYYY/MM/DD
   概要: ...

2. [ページタイトル] — 最終更新: YYYY/MM/DD
   概要: ...

詳細を確認するページ番号をお知らせください。
```

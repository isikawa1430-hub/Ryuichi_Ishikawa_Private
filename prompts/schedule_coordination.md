# 日程調整テンプレート

## 使い方
「〇〇さんとの打ち合わせを来週入れたい。1時間程度」などと指示してください。

## エージェントの実行手順

1. `list_events` で指定期間の予定を取得（アカウント：ryuichi.ishikawa@gmail.com）
2. 空き時間を抽出（デフォルト：営業時間 9:00-18:00）
3. 候補日時を3〜5つ提示
4. ユーザーが選択したら `create_event` で予定を作成

## 確認事項

- 対象期間（来週・今月中 など）
- 所要時間
- 参加者（わかる場合）
- 優先する時間帯（午前・午後 など）
- オンライン/対面の別

## 出力フォーマット

```
空き候補（〇〇さんとの打ち合わせ / 1時間）：

1. MM/DD(曜) HH:MM〜HH:MM
2. MM/DD(曜) HH:MM〜HH:MM
3. MM/DD(曜) HH:MM〜HH:MM

ご希望の候補番号をお知らせください。
予定のタイトルと参加者メールアドレスもご確認ください。
```

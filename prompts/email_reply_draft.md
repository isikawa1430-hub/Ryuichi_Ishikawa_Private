# メール返信下書きテンプレート

## 使い方
「[キーワードまたは件名]のメールに返信する下書きを作成して」と指示してください。

## エージェントの実行手順

1. `search_threads` で対象メールを特定
2. `get_thread` でスレッド全体を読み込み、文脈を把握
3. 以下の情報を確認してから下書きを作成：
   - 送信者の名前・会社名
   - メールのトーン（丁寧・カジュアル等）
   - 返答すべき質問・アクション
4. `create_draft` で下書きを保存

## 下書き作成ルール

- 宛名は「〇〇様」形式
- 送受信アカウント：Ryuichi's.ishikawa@gmail.com
- 自分の署名：石川隆一 / 株式会社アズサ設計
- 件名は元の件名に「Re: 」を付ける
- 具体的な返答内容はユーザーに確認してから記入
- 曖昧な部分は「〇〇」でプレースホルダーを使う

## 出力例

```
宛先: xxx@example.com
件名: Re: 〇〇についてのご確認

〇〇様

お世話になっております。
アズサ設計の石川でございます。

[返信内容]

何卒よろしくお願いいたします。

石川隆一
株式会社アズサ設計
```
